Aide

Rôle par défaut : Définit le rôle auquel ce template sera automatiquement assigné.

Permissions : Cochez les cases pour activer les permissions correspondantes.

Exemples :

  • Utilisateur standard : Voir tableau de bord, consulter rapports, voir directions
  • Manager : + Gérer risques, KRI, exporter données, voir liste utilisateurs
  • Auditeur : + Gérer audits, plans d'action, voir directions
  • Admin client : Toutes les permissions sauf gérer clients/provisionner serveurs
  • Super Admin : Toutes les permissions

Nouvelles permissions :

  • can_view_departments : Voir les directions et services
  • can_manage_departments : Créer/modifier directions et services
  • can_view_users_list : Voir la liste des utilisateurs
  • can_edit_users : Modifier les utilisateurs (sauf admin client)
