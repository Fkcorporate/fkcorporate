Guide d'utilisation
Comment configurer les champs :
  1. Définissez le nom technique (unique)
  2. Choisissez le type de champ approprié
  3. Organisez par section pour une meilleure lisibilité
  4. Définissez si le champ est obligatoire
  5. Utilisez le glisser-déposer pour l'ordre d'affichage
Bonnes pratiques
  • Utilisez des noms techniques sans espaces
  • Préférez les listes déroulantes pour les choix limités
  • Ajoutez un texte d'aide pour guider l'utilisateur
  • Testez les champs après leur création
  • Limitez les champs obligatoires au strict nécessaire
